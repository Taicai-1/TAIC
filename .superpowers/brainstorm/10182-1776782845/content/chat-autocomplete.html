Comportement

  • Le menu apparaît dès que l'utilisateur tape /
  • Le menu filtre en temps réel (ex: /ana → montre seulement /analyse)
  • Navigation au clavier : ↑ ↓ pour sélectionner, Entrée pour valider
  • Clic sur une option → remplit le champ avec la commande et envoie directement
  • Seules les commandes assignées à ce companion sont affichées
  • Escape ou effacer le / ferme le menu
